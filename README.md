# UNRIVAL App

The fixtures listed below must have their firmware updated before they will work with the UNRIVAL app.

## How It Works

Getting a fixture working with the UNRIVAL app is a two-step process:

1. **Update the firmware.** Update the fixture to the relevant firmware listed in the tables below.
2. **Scan and check the tag.** In the app, use **"Scan a fixture"** to scan the tag. You will see **"Invalid Tag"** — click **"Check Tag"** and follow the on-screen instructions.

> [!NOTE]
> Follow your fixture's user manual for the correct firmware installation procedure.
